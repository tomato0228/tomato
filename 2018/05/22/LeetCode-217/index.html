存在重复元素 Contains Duplicate

给定一个整数数组，判断是否存在重复元素。
如果任何值在数组中出现至少两次，函数返回 true。如果数组中每个元素都不相同，则返回 false。

解决思路

定义一个set，遍历数组，如果set里面没有当前数，则把当前数放入set中，否则返回true，遍历完后，说明没有重复的则返回false。
